Declare TODAY'S stop condition in one sentence. Format:

> Today's goal: <verb> <object> + <quality criterion> + push to main, then STOP for operator review.

Example:
> Today's goal: implement Tier 0 Macro Dashboard data layer (ECOS + FRED clients + composite scoring + caching + types) + /dashboard page shell with placeholder data wiring + push to main, then STOP for operator review.

Do NOT start sub-tasks until the user confirms the goal. If the goal is too large for one Day session (>~4-6 hours of work), propose splitting and ask.
