# 포트폴리오 도구 — 4단 실행 로드맵 + 단·중·장기

> **제품 정체성 (확정 방향):** 구독형 **소프트웨어**. 사용자 본인 정보·본인 브로커(BYOK)·본인 IPS. 코호트는 **자금 비수탁**, **투자자문·일임 아님** (Option B).  
> **법률 자문 아님** — vault 22·약관과 함께 보수적 설계.

---

## 4단 실행 사다리 (기능 깊이)

| 단계 | 이름 | 사용자 경험 | 규제·설계 경계 |
|------|------|-------------|----------------|
| **L1** | **입력·기록** | 수동 비중·종목, CSV, 온보딩 Q2 JSONB, IPS 위저드(Phase 2) | 데이터 주체 = 사용자. **서비스가 비중을 “정해 주지” 않음** |
| **L2** | **조회·거울** | 브로커 API **읽기 전용**(잔고·체결), 목표 대비 drift, 회전율·처분효과 | BYOK·암호화 저장. **주문 API 호출 없음** |
| **L3** | **알림·페이스** | Shape C 트리거, “본인 규칙 발동”, 분할매수 페이스 대비 상태 | **신호만** — 추천/권장/타이밍 금지 (Option B) |
| **L4** | **승인 후 실행** | “이 규칙이면 이 주문 묶음” **프리뷰** → 사용자 **1-tap 승인** → BYOK `placeOrder` | **무인 자동매매는 기본값 아님**. 멱등성·한도·감사 로그 (ADR) |

**금지 구간 (모든 단계):** AI/서비스가 목표 비중·매매 타이밍을 **처방**, 플랫폼 공용 키로 **타인 계좌 대리 매매**, 수익률 비교·리더보드.
